# Career Stack DB

Career Stack DB is a full-stack career management platform designed to help users organize, track, and manage their professional growth in one place.

The application allows users to:
- Create and manage career profiles
- Track skills and technologies
- Store project and experience information
- Manage resumes and career-related data
- Organize development progress through a structured database system

## Features

- User Authentication and Authorization
- Career Profile Management
- Skills Tracking System
- Project and Experience Database
- Modern Responsive Frontend
- Backend API Integration
- Database-Driven Architecture

## Tech Stack

### Frontend
- React.js
- HTML/CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
